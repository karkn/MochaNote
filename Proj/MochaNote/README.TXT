MochaNote
mocha
http://www.confidante.jp/


■ はじめに

MochaNoteは高い記述力，柔軟な管理機能，強力な検索機能を持った
ノート管理ソフトです。思いつきや調べ物などをノートに残して，
後で見直して思い出したり整理したりできます。 

■ 動作環境

以下の環境で動作します。

  ・Windows XP / Vista / 7 / 10 日本語版
  ・.NET Framework 3.5 SP1

Windows XP SP3，.NET Framework 3.5 SP1が推奨環境です。

■ファイルの内容

  ・setup.exe
      Confidanteインストーラです。
      Confidanteをインストールするにはこのファイルを実行してください。
  ・Confidante.msi
      setup.exeから呼び出されます。
      Confidante.msiではなくsetup.exeを使ってインストールしてください。
  ・RELEASE.TXT
      リリースしたバージョンごとの機能の追加・修正の内容を書いた
      テキストファイルです。
  ・README.TXT
      このファイルです。

■ インストール

管理者レベルのユーザでログオンしてsetup.exeを実行してください。
インストーラが起動しますので画面の指示に従ってインストールしてください。
.NET Framework 3.5 SP1が入っていない場合はConfidanteのインストールの前に
.NET Framework 3.5 SP1のインストールが始まります。

■ アンインストール

コントロールパネルの「アプリケーションの追加と削除」から，
「Confidante」を選んで削除して下さい。
ノートデータファイルやライセンスファイルはアンインストールでは削除されません。
必要であれば手動で削除してください。

■ 使用方法

ヘルプにチュートリアルを載せてありますので，最初にご覧ください。
一通り見て実際に操作してみればConfidanteの使い方がつかめると思います。
なお，ヘルプはメインメニューの[ヘルプ]>[Confidanteヘルプ]を選択すると
Webブラウザで表示されます。

■ フリーライセンス

Confidanteは無料でずっと利用することができます。
フリーライセンスで利用するのにユーザ登録などの作業は必要ありません。
