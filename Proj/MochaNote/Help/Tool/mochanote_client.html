MochaNoteClient

MochaNoteClientはMochaNoteを操作するためのコマンドラインアプリケーションです。 MochaNoteClientにコマンドライン引数を渡すことでノートの作成，ロード，削除といった処理を呼び出せます。

使い方は以下のUsageのとおりです。 

Usage:
  MochaNoteClient.exe <method and args> [options]

  <method and args> are
    create <title>: create a new memo
    load <memoUri>: load a memo
    remove <memoUri>: remove a memo
    uris: query all memo URIs
    exist <memoUri>: query existence of a memo
    title <memoUri>: query a title of a memo

    <title> is a string that is a title of a creating memo
    <memoUri> is a string that was returned on creating or querying memo URIs

  [options] are
    -a: activate MochaNote window
    -s[=seconds]: start MochaNote if not started and wait for seconds.
                  If seconds omitted, wait for 20 seconds

  Example:
    MochaNoteClient.exe create "My memo"
      create a memo titled "My memo"
    MochaNoteClient.exe load MEMO_URI
      load a memo specified by MEMO_URI
    MochaNoteClient.exe create "My memo" -s=30 -a
      start MochaNote if not started, and create a memo, and activate
