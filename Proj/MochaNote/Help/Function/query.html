ノートを検索する

ノートを検索するには以下の操作を行います。

  1. 「ワークスペース」ペインの「検索」テキストボックスに検索する文字列を入力する
    全文検索ではなく他の条件を使って検索する場合は「検索」テキストボックスに何も入力しなくてもかまいません。
  2. Enterキーを押すか，「検索」アイコンをクリックする
    「ワークスペース」ペインの表示が切り替わり「クエリビルダ」が表示されます。 検索文字列を入力していた場合は，その文字列を含むノートが「ノートリスト」ペインに表示されます。
  3. クエリビルダでその他の条件を設定する
    必要があればクエリビルダの「タイトル」・「タグ」・「マーク」・「期間」といった条件を設定して 「ノートリスト」ペインに表示されるノートを絞り込みます。
  4. 「検索」テキストボックスでCtrl+Enterキーを押すか「×」ボタンをクリックして検索を終了する
    検索を終了すると「ワークスペース」ペインの表示が切り替わり，ワークスペースツリービューが表示されます。
