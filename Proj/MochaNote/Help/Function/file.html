ファイルを埋め込む

MochaNoteではノートにファイルを埋め込んだり，ショートカットを貼り付けることができます。

ファイルを埋め込む

埋め込み元のファイルをコピーしてノートにファイルを埋め込みます。 埋め込んだファイルはコピーされているので元のファイルの更新や削除や移動の影響を受けなくなります。

ファイルを埋め込むには以下のいずれかの操作を行います。

  • メインツールバーの「ファイルを追加」>「ファイル」をクリックしてファイルを選択する
  • エクスプローラからファイルをドラッグ&ドロップする
  • エクスプローラでファイルをコピーしてからメインメニューの「編集」>「貼り付け」を選択する

ファイルのショートカットを貼り付ける

ファイルのショートカットをノートに貼り付けます。 貼り付けたショートカットは元のファイルの更新や削除や移動の影響を受けます。 ショートカット元のファイルが更新されてもそのファイルを最新の状態で開くことができますが，削除や移動をされるとファイルを開けなくなります。

ファイルのショートカットを貼り付けるには以下のいずれかの操作を行います。

  • メインツールバーの「ファイルを追加」>「ファイルのショートカット」をクリックしてファイルを選択する
  • エクスプローラからファイルをCtrlキーを押しながらドラッグ&ドロップする

フォルダのショートカットを貼り付ける

フォルダのショートカットをノートに貼り付けます。

フォルダのショートカットを貼り付けるには以下のいずれかの操作を行います。

  • メインツールバーの「ファイルを追加」>「フォルダのショートカット」をクリックしてフォルダを選択する
  • エクスプローラからフォルダをドラッグ&ドロップする

ファイルを取り出す

埋め込んだファイルはノートの外に取り出すことができます。

ノートを取り出すには以下の操作を行います。

  1. ノート内のファイルを右クリックしてコンテキストメニューの「取り出す」を選択する
  2. ダイアログでファイルの保存先を選択する

自動インポート機能を使う

自動インポート機能を使うには以下の操作を行います。

  1. マイドキュメント下の「MochaNote\Import」フォルダにファイルをコピーする
    ファイルをコピーするとノートに自動的に埋め込まれます。
