ノート内を検索する

ノート内の文字列を検索する

ノート内の文字列を検索するには以下の操作を行います。

  1. メインツールバーの「ノート内検索」をクリック，またはメインメニューの「編集」>「ノート内検索」を選択する
    検索パネルが表示されます。
  2. 検索パネルの「検索」テキストボックスに検索する文字列を入力する
    文字列を入力するとその文字列が含むノート要素が選択状態になります。 文字列を含むノート要素がテキストの場合，テキストが編集状態になり検索された文字列の箇所が選択状態になります。
  3. 検索を終了する
    EnterキーまたはEscキーを押すと検索を終了して検索パネルが非表示になります。 Enterキーを押した場合は終了後も検索時の選択状態のままになります。 Escキーを押した場合は終了後は検索開始前の選択状態に戻ります。

ノート内の文字列を置換する

ノート内の文字列を置換するには以下の操作を行います。

  1. メインツールバーの「ノート内検索」をクリック，またはメインメニューの「編集」>「ノート内検索」を選択する
    検索パネルが表示されます。
  2. 検索パネルの「検索」テキストボックスに検索する文字列を入力する
    文字列を入力するとその文字列が含むノート要素が選択状態になります。 文字列を含むノート要素がテキストの場合，テキストが編集状態になり検索された文字列の箇所が選択状態になります。 テキストが編集状態になった場合のみ検索パネルの「置換」テキストボックスに文字列を入力可能になります。
  3. 検索パネルの「置換」テキストボックスに置換する文字列を入力して「置換」テキストボックス右の矢印ボタンをクリックする
    現在選択状態の文字列が「置換」テキストボックスの文字列に置換したあと， その位置から前方または後方に「検索」テキストボックスの文字列を検索します。
  4. 置換を終了する
    EnterキーまたはEscキーを押すと置換を終了して検索パネルが非表示になります。 Enterキーを押した場合は終了後も検索時の選択状態のままになります。 Escキーを押した場合は終了後は検索開始前の選択状態に戻ります。
