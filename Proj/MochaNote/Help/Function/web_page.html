Webページを貼り付ける

Internet ExplorerやFirefoxといったWebブラウザからWebページの一部をノートに貼り付けることができます。

なお，表の中の表や表の中の画像には対応していません。

ドラッグ&ドロップで貼り付ける

Webページをドラッグ&ドロップで貼り付けるには以下の操作を行います。

  1. Webブラウザで貼り付ける箇所を選択状態にする
  2. WebブラウザからMochaNoteのノートエディタにドラッグ&ドロップする

クリップボードで貼り付ける

Webページをクリップボードで貼り付けるには以下の操作を行います。

  1. Webブラウザで貼り付ける箇所を選択状態にする
  2. Webブラウザで「コピー」を選択する
  3. MochaNoteのメインメニューの「編集」>「貼り付け」を選択する

クリップ機能で貼り付ける

クリップ機能で貼り付けるには以下の操作を行います。

  1. Webブラウザで貼り付ける箇所を選択状態にする
  2. Ctrl+Alt+Cキーを押すか，タスクトレイのMochaNoteアイコンを右クリックして「ノートにクリップ」を選択する

クリップ機能でWebページを貼り付けた場合，新しく作られたノートの情報元にはWebページのアドレスが自動的に設定されます。
